# Chai-Fi

## Description

Tool to make adding public WiFi networks to wpa_supplicant.conf easier.

## Build

```
go get
go build
```

## Usage
```
Usage of ./chaifi:
  -f string
    	path to wpa_supplicant.conf (default "/etc/wpa_supplicant.conf")
  -i string
    	wireless interface (default "wlan0")
```
